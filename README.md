O meu objetivo foi entregar algo funcional em pouco tempo, apesar dos espaços
entre os commit's o projeto todo levou uma media de 8 horas de trabalho. Contando
com a configuração do servidor.


## Instalação Laravel

```sh
cd back
composer install
php artisan serve
```

## Instalação React

```sh
cd front
yarn install
yarn start
```
